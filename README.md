# 📰 News Scraper – Últimas Notícias do Mercado

Script em Python que captura automaticamente as **últimas notícias do mercado financeiro** no site Dados de Mercado e permite abrir qualquer notícia diretamente no navegador pelo terminal.

Projeto simples, prático e ideal para quem acompanha:

- 📊 Mercado financeiro  
- 📈 Notícias econômicas  
- 🤖 Automação com Python  
- 🧠 Estudos de Web Scraping  

---

## 🌐 Fonte dos Dados

As notícias são extraídas de:

https://www.dadosdemercado.com.br/ultimas-noticias

---

## 🚀 Funcionalidades

✅ Captura automática das últimas 20 notícias  
✅ Exibe títulos no terminal  
✅ Mostra horário da última atualização  
✅ Permite atualizar as notícias manualmente  
✅ Abre a notícia escolhida no navegador  
✅ Interface simples e interativa  

---

## 🛠️ Tecnologias Utilizadas

- Python 3.x  
- `requests` – Requisições HTTP  
- `beautifulsoup4` – Parsing de HTML  
- `datetime` – Controle de horário  
- `webbrowser` – Abertura automática do link
